# PRD 005: Future Direction

## Purpose

Define future product direction for `X-Port`.

## Explicitly Deferred

- browser extension for capturing bookmarks
- hosted sync service
- desktop wrapper or bundled app shell
- cross-platform account sharing
- AI enrichment, classification, or semantic search in V1

## Likely Future Directions

- incremental sync
- scheduled exports
- better search over local archives
- Continuum integration for downstream knowledge workflows
- optional richer presentation layer if the CLI becomes insufficient

## Guardrails

- future features must not weaken the local-first model
- future features must not require shared credentials
- future features should preserve export format stability

## Acceptance Criteria

- future work is documented without being folded into V1 scope
- the current repo can stay small while leaving room for expansion
