# X-Port

X-Port is a local-first CLI for exporting a user's X bookmarks into durable formats.

## Current State

- docs framework bootstrapped
- initial CLI/package scaffold in place
- export formatters available as pure functions

## Commands

- `xport login`
- `xport export --format md`
- `xport export --format json`
- `xport export --format csv`

## Development

```bash
npm install
npm run build
npm test
```
